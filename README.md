# Barber Shop Appointment App 💇‍♂️✂️

تطبيق متكامل لإدارة حجوزات صالونات الحلاقة، تم تطويره باستخدام إطار العمل **Flutter**. يهدف التطبيق إلى تحويل عملية الحجز التقليدية إلى تجربة رقمية سلسة، مما يوفر الوقت على العميل وينظم جدول العمل للحلاق.

## 🎯 الهدف من المشروع

تقديم حل تقني يجمع بين الأناقة في التصميم (UI) والفعالية في الأداء (UX)، حيث يتيح للزبائن اختيار الخدمات، الحلاق المفضل، والوقت المناسب بضغطات بسيطة.

## ✨ المميزات التفصيلية (Detailed Features)

### 1. تجربة مستخدم مخصصة (User Experience)

* **شاشة ترحيبية (Onboarding):** واجهة عصرية تشرح مميزات التطبيق للمستخدم الجديد.
* **لوحة تحكم المستخدم (Dashboard):** عرض سريع للخدمات المتاحة والعروض الحالية.
* **نظام اختيار الخدمات:** قائمة منسدلة أو بطاقات تفاعلية تتيح اختيار نوع الحلاقة (شعر، ذقن، عناية بالبشرة) مع عرض السعر والمدة الزمنية المتوقعة لكل خدمة.

### 2. نظام الحجز الذكي (Booking System)

* **تقويم تفاعلي (Interactive Calendar):** اختيار اليوم والتاريخ مع تحديث فوري للأوقات المتاحة.
* **إدارة الفترات الزمنية (Time Slots):** تقسيم اليوم إلى فترات محددة لمنع تداخل المواعيد.
* **تأكيد الحجز:** ملخص كامل للطلب قبل التأكيد النهائي.

### 3. التصميم والهوية البصرية

* **Dark & Light Theme:** دعم كامل للوضع الليلي والنهاري بما يتناسب مع ذوق المستخدم.
* **Custom Icons & Graphics:** استخدام أيقونات مخصصة تعكس طبيعة العمل (مقص، مشط، إلخ).

## 🛠 البنية التقنية (Technical Architecture)

تم بناء التطبيق باتباع أفضل الممارسات في تطوير تطبيقات الفلاتر:

* **Flutter Framework:** لضمان أداء عالٍ (Native Performance) على Android و iOS.
* **State Management (Provider/GetX):** لإدارة حالة التطبيق بكفاءة، خاصة عند تحديث المواعيد والخدمات المختارة.
* **Clean Code Principles:** تقسيم الكود إلى ملفات منفصلة (Views, Controllers, Models) لسهولة الصيانة والتطوير المستقبلي.
* **Responsive Design:** الواجهات مصممة لتعمل بانسجام على مختلف أحجام الشاشات.

## 📂 تقسيم المجلدات (Directory Structure)

```text
lib/
├── core/          # الألوان، الثيمات، والثوابت
├── models/        # نماذج البيانات (Service, Booking, User)
├── views/         # الشاشات الرئيسية (Home, Booking, Profile)
├── widgets/       # العناصر البرمجية المشتركة (Buttons, Cards)
└── controllers/   # المنطق البرمجي ومعالجة البيانات

```

## 🚀 التشغيل والتثبيت (Setup)

1. قم بتحميل المستودع:
```bash
git clone https://github.com/salem-baqatyan/barber_app.git

```


2. تثبيت الحزم البرمجية:
```bash
flutter pub get

```


3. تشغيل المشروع:
```bash
flutter run --release

```
